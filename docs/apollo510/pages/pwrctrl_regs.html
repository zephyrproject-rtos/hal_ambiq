MCUPERFREQ - Performance control for MCU

Address:

  Instance 0 Address:   0x40021000

Description:

This register provides the performance mode knobs for MCU. S/w should write the *PERFREQ field to desired mode and wait for the *PERFACK and check for the *PERFSTATUS. Some times system may not allow certain modes but *PERFACK should always follow *PERFREQ change.

MEMPWREN - Enables individual banks of the MEMORY array

Address:

  Instance 0 Address:   0x40021014

Description:

Enables individual banks of memories. When a PWREN field is set, power is enabled to the memory or specific bank of memory. This register works in conjunction with the MEMRETCFG register. If a field in this register is not set, then power is always disabled to the memory bank. In the case of NVM, to enable NVM1, NVM0 must also be enabled. Disabling power to NVM0 also disables power to NVM1.

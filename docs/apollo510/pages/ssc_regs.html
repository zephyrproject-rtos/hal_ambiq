Bits Name RW Description
31:11 RSVD RO RESERVED
10 LOCKSSRAMPROT RW Write 1 to disables writes to the SSRAMxPROT register from software, any master, or from any debug agent. Once set, it remains locked until the next system reset. Write 0 has no effect.
9 LOCKSVTAIRCR RW Write 1 to disable writes to the following secure registers from software or from a debug agent that is connected to the processor: VTOR_S/AIRCR.PRIS/AIRCR.BFHFNMINS. Write 0 has no effect.
8 LOCKNSVTOR RW Write 1 to disable writes to the VTOR_NS register. Write 0 has no effect.
7 LOCKSMPU RW Write 1 to disable writes to registers that are associated with the Secure Memory Protection Unit (MPU) region from software or from a debug agent connected to the processor. Write 0 has no effect.
6 LOCKNSMPU RW Write 1 to disable writes to registers that are associated with the Non-secure MPU region from software or from a debug agent connected to the processor. Write 0 has no effect.
5 LOCKSAU RW Write 1 to disable writes to registers that are associated with the SAU region from software or from a debug agent connected to the processor. Write 0 has no effect.
4 LOCKTCM RW Write 1 to disable writes to registers that are associated with the TCM region from software or from a debug agent connected to the processor. Write 0 has no effect.
3 LOCKITGU RW Write 1 to disable writes to registers that are associated with the ITCM interface security gating from software or from a debug agent connected to the processor. Write 0 has no effect.
2 LOCKDTGU RW Write 1 to disable writes to registers that are associated with the DTCM interface security gating from software or from a debug agent connected to the processor. Write 0 has no effect.
1 LOCKPAHB RW Write 1 to disable writes to the PAHBCR register from software or from a debug agent connected to the processor. Write 0 has no effect.
0 LOCKDCAIC RW Write 1 to disable access to the instruction cache direct cache access registers DCAICLR and DCAICRR. Write 0 has no effect.
